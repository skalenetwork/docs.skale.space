The **Interchain Messaging Agent (IMA)**, also known as the **SKALE IMA Bridge**, enables seamless token and message transfers between Ethereum and SKALE Chains.

IMA is **flexible**, **modular**, **secure**, and **cost-efficient**, making it a powerful tool for developers building cross-chain dApps.

## Features

- **Chains:** Supports bridging `Ethereum` ↔ `SKALE Chains` and `SKALE Chains` ↔ `SKALE Chains`

- **Messagin Protocol:** It supports any kind of messages and token standards (ex: ETH, ERC-20, ERC-721, ERC-1155, etc).

- **Speed:** Very fast bridge taking between 30 to 60 seconds to complete the bridging.

- **Cost Efficient:** The transfer between SKALE Chains (S2S) are free and the costs for Ethereum transfers are low.
