The SKALE Network is a multi-chain network where "owning" or "operating" a SKALE Chain
does not require the owner or operator to actually run the nodes. This is because the SKALE Network
has validators to run the SKALE Chains.

The operator(s) of a SKALE Chain have special privleges that enable them to manage the chain and provide
access and resources to others; whether in their team for a dedicated AppChain or for a 3rd party if a HubChain.
