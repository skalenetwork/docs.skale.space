## Hardware Requirements

The following are the hardware requirements for running a SKALE Network validator node:

- **Machine**: A Linux x86_64 machine
- **Physical Cores**: At least 8 physical cores
- **RAM Size**: At least 32 GB RAM
- **Attached Storage**: Separate not mounted block device - 2TB (1.9 TB actual measure)
- **Swap Size**: At least 16 GB swap
- **Port Range**: See networking requirements in compliance documentation

For more detailed hardware requirements and compliance information, see [Compliance Requirements](/validators/compliance-requirements).
