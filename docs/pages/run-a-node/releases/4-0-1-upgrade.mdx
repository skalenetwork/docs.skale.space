# SKALE 4.0.1 Upgrade

#### Upgrade steps for the node server

::::steps


1. Update the following parameters to the new values

    ```bash
    CONTAINER_CONFIGS_STREAM=4.0.1
    ```


2. Run update procedure
    ```bash
    skale node update .env --yes
    ```
