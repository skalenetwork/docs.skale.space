# SKALE 3.0.3 Upgrade

## Upgrade steps from 3.0.2 to 3.0.3

::::steps

1. Update the following parameters to the new values

    ```bash
    CONTAINER_CONFIGS_STREAM=3.0.3
    ```

2. Execute update

    ```bash
    skale node update .env --yes
    ```
