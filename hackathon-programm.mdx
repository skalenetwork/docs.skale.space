## Start building 

### Privacy SDK

The Privacy SDK enables developers to build applications with programmable privacy, including:

- Confidential token transfers
- Encrypted transactions
- Private balance management
- Re-encryption workflows

Start building privacy-first applications on SKALE and unlock new possibilities for Web3.
