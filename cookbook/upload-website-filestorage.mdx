## Upload a Website to Filestorage

This tutorial will show you how to upload a website to SKALE's decentralized filestorage.

## Prerequisites

- Node.js and npm installed
- A wallet with sFUEL
- Website files ready

## Steps

1. Install filestorage library
2. Prepare your website files
3. Upload to filestorage
4. Access your website
