## Overview

SKALE Network is governed by the SKALE DAO, which allows SKL token holders to participate in network decisions.

## Governance Model

- **SKALE DAO** - Decentralized autonomous organization managing the network
- **Proposals** - Community-submitted proposals for network changes
- **Voting** - SKL token holders vote on proposals
- **Execution** - Approved proposals are executed on-chain

## Participation

- Hold and delegate SKL tokens
- Submit governance proposals
- Vote on active proposals
- Participate in discussions
