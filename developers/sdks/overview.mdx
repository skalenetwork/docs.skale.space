SKALE provides official SDKs and libraries to help developers build on the network. These SDKs are maintained by the SKALE team and provide first-class support for SKALE's unique features.

## Available SDKs

### Filestorage.js

The official JavaScript SDK for SKALE's native onchain filestorage feature. Store files and websites directly on SKALE Chains.

## Installation

Each SDK can be installed via its respective package manager:

```bash
# Filestorage.js
npm install @skalenetwork/filestorage.js@~1.0.1
```

## Getting Started

1. Choose the SDK that matches your needs
2. Install the SDK using the package manager
3. Follow the SDK-specific documentation
4. Check out the examples in the [Cookbook](/cookbook)
