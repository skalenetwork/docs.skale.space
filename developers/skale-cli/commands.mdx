Complete reference for all SKALE CLI commands. Each command supports `--help` for detailed usage information.

## Chain Management (`chains`)

Aggregate portfolio, compare balances, and list supported chains.

### `chains list`
List all supported chains with current status.

```bash
skale chains list
skale chains list --network skale
skale chains list --status
```

### `chains portfolio <address>`
Aggregate token balances across multiple chains.

```bash
skale chains portfolio 0xAb5801a7D398351b8bE11C439e05C5B3259aeC9B
skale chains portfolio 0x... --chains skale-base,titan
skale chains portfolio 0x... --token 0x... --value-in USD
```

**Options:**
- `--token`: Token contract address for ERC20 balance
- `--chains`: Comma-separated chain names (e.g., 'skale-base,titan,mainnet')
- `--value-in`: Display values in: USD, EUR, ETH

### `chains compare <address>`
Compare address balance across chains.

```bash
skale chains compare 0x... --chains skale-base,titan,mainnet
skale chains compare 0x... --format table
```

## Contract Reading (`read`)

Call read-only contract methods on SKALE or Ethereum.

```bash
# SKALE chain
skale read configController version --chain skale-base
skale read configController isAddressWhitelisted 0x... --chain skale-base

# Ethereum mainnet
skale read sklToken totalSupply --network mainnet
skale read sklToken balanceOf 0x... --network mainnet
```

**Valid Contracts:**
- **SKALE chains:** `configController`, `messageProxyForSchain`, `tokenManagerERC20`
- **Ethereum:** `messageProxy`, `sklToken`

## IMA Bridge (`ima`)

Interchain Messaging Agent (IMA) bridge operations.

### `ima chain-id`
Get chain ID from MessageProxy contract.

```bash
skale ima chain-id --chain skale-base
skale ima chain-id --network mainnet
```

### `ima connected-chains`
Check if a chain is connected to MessageProxy.

```bash
skale ima connected-chains --chain skale-base --check-chain mainnet
```

### `ima deposit`
Get deposit information for ERC20 tokens.

```bash
skale ima deposit --chain skale-base --network mainnet --token 0x...
```

### `ima withdraw`
Get withdraw information for ERC20 tokens.

```bash
skale ima withdraw --chain skale-base --token 0x...
```

## BITE Encryption (`bite`)

Blockchain Integrated Threshold Encryption (BITE) commands. Available on `skale-base`, `skale-base-sepolia`, and `skale-bite-sandbox` chains.

### `bite encrypt-tx`
Encrypt a transaction using BITE.

```bash
skale bite encrypt-tx --chain skale-base --to 0x... --data 0x...
```

### `bite decrypt-tx`
Decrypt transaction data from a tx hash.

```bash
skale bite decrypt-tx --chain skale-base --txhash 0x...
```

### `bite encrypt-msg`
Encrypt a message using BITE.

```bash
skale bite encrypt-msg --chain skale-base --message "Hello World"
```

### `bite info`
Get BITE protocol info for a chain.

```bash
skale bite info --chain skale-base
```

## Transaction Preparation (`txprep`)

Prepare and encode transaction data.

### `txprep encode`
Generate calldata for contract method calls.

```bash
skale txprep encode 'transfer(address,uint256)' '0x...,100'
```

### `txprep selector`
Get function selector for a method signature.

```bash
skale txprep selector 'transfer(address,uint256)'
```

### `txprep encode-param`
Encode parameters for smart contract calls.

```bash
skale txprep encode-param address 0x...
skale txprep encode-param uint256 1000
skale txprep encode-param string 'hello world'
```

### `txprep build-tx`
Build unsigned transaction object.

```bash
skale txprep build-tx 0x... 100 --eth
skale txprep build-tx 0x... 1000000000000000000 --chain calypso --data 0xa9059cbb...
```

## Wallet Operations (`wallet`)

Wallet management and address utilities.

### `wallet balance <address>`
Get ETH or token balance for an address.

```bash
skale wallet balance 0x... --chain skale-base
skale wallet balance 0x... --network mainnet --token 0x...
```

### `wallet checksum <address>`
Convert address to checksummed format.

```bash
skale wallet checksum 0xab5801a7d398351b8be11c439e05c5b3259aec9b
```

### `wallet generate`
Generate a new Ethereum wallet (with warnings).

```bash
skale wallet generate
skale wallet generate --password mypassword
```

### `wallet recover`
Recover wallet from private key or mnemonic (with warnings).

```bash
skale wallet recover --private-key 0x...
skale wallet recover --mnemonic "word1 word2 word3 ..."
```

### `wallet validate <address>`
Validate Ethereum address format and checksum.

```bash
skale wallet validate 0xAb5801a7D398351b8bE11C439e05C5B3259aeC9B
```

### `wallet ens <name>`
Resolve ENS name to address or perform reverse resolution.

```bash
skale wallet ens vitalik.eth
skale wallet ens 0xAb5801a7D398351b8bE11C439e05C5B3259aeC9B
```

### `wallet address <privateKey>`
Generate Ethereum address from a private key without exposing it.

```bash
skale wallet address 0x...privateKey...
```

### `wallet explorer <address>`
Generate explorer URL for an address or transaction.

```bash
skale wallet explorer 0x... --chain skale-base
```

## Gas (`gas`)

Get gas price and estimates.

### `gas price`
Get current gas price with speed estimates.

```bash
skale gas price --chain skale-base
```

### `gas estimate`
Estimate gas for a transaction.

```bash
skale gas estimate --to 0x... --value 1 --eth --chain skale-base
```

### `gas history`
Get historical gas prices for a chain.

```bash
skale gas history --chain skale-base
```

### `gas simulate`
Simulate a transaction without executing it.

```bash
skale gas simulate --to 0x... --value 1 --eth --chain skale-base
skale gas simulate --to 0x... --data 0xa9059cbb... --chain skale-base
```

## Global Options

All commands support these global options:

| Option | Description |
|--------|-------------|
| `--help, -h` | Show help for the command |
| `--version, -v` | Show CLI version |
| `--format` | Output format: json, text, table (default: text) |
| `--chain` | SKALE chain name |
| `--network` | Ethereum network: mainnet, sepolia |

## Output Formats

The CLI supports multiple output formats:

- **text**: Human-readable output (default)
- **json**: Structured JSON for programmatic use
- **table**: Tabular format for easy reading

Example:
```bash
skale chains list --format json
skale chains portfolio 0x... --format table
```
