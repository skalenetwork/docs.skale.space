Returns information about an uncle block by hash and uncle index position.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getUncleByBlockHashAndIndex",
  "params": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "0x0"
  ],
  "id": 1
}
```

### Parameters

1. **blockHash** (`string`, required): Block hash, 32 bytes hex-encoded with `0x` prefix
2. **uncleIndex** (`string`, required): Uncle index position, hex-encoded

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": null
}
```

### Response Types

- **result** (`object` | `null`): Uncle block object or `null` if not found

## Notes

- Always returns `null` - SKALE does not have uncles
- Available on all node types
- SKALE uses Proof of Stake consensus, not Proof of Work
