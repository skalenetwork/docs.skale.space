<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getFilterLogs" />


Returns an array of all logs matching filter with given id.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getFilterLogs",
  "params": [
    "0x1"
  ],
  "id": 1
}
```

### Parameters

1. **filterId** (`string`, required): Filter ID, hex-encoded

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": [
    {
      "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
      "topics": ["0x..."],
      "data": "0x...",
      "blockNumber": "0x1a2b3c",
      "blockHash": "0x...",
      "transactionHash": "0x...",
      "transactionIndex": "0x0",
      "logIndex": "0x0",
      "removed": false
    }
  ]
}
```

### Response Types

- **result** (`array`): Array of log objects

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",

```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": []
}
```

</CodeGroup>

## Notes

- Available on all node types
- Returns all logs matching the filter, not just new ones
- Use [eth_getFilterChanges](/api-reference/methods/eth-getFilterChanges) for incremental updates
