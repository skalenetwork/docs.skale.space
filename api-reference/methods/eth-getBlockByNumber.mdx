<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getBlockByNumber" />


Returns information about a block by block number.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockByNumber",
  "params": [
    "latest",
    true
  ],
  "id": 1
}
```

### Parameters

1. **block** (`string`, required): Block number or one of `"latest"`, `"earliest"`, `"pending"`
2. **fullTransactions** (`boolean`, required): If `true`, returns full transaction objects; if `false`, returns only transaction hashes

### Response Types

- **number** (`string`): Block number, hex-encoded
- **hash** (`string`): Block hash, 32 bytes hex-encoded
- **parentHash** (`string`): Hash of the parent block
- **transactionsRoot** (`string`): Root of the transaction trie
- **stateRoot** (`string`): Root of the state trie
- **gasLimit** (`string`): Maximum gas allowed in this block
- **gasUsed** (`string`): Total gas used by all transactions in this block
- **timestamp** (`string`): Unix timestamp of when the block was mined
- **transactions** (`array`): Array of transaction objects or transaction hashes (depending on `fullTransactions` parameter)

## Notes

- Use `"latest"` to get the most recent block
- Set `fullTransactions` to `false` for faster responses when you only need transaction hashes
- SKALE blocks are produced approximately every 1 second
- This method is available on all node types
