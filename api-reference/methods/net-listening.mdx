<Api openapi="api-reference/openapi.json" path="/" method="post" example="net_listening" />


Returns `true` if client is actively listening for network connections.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "net_listening",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```

### Response Types

- **result** (`boolean`): `true` if listening, `false` otherwise

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "net_listening",
    "params": [],
    "id": 1
  }'
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```

</CodeGroup>

## Notes

- Available on all node types
- Typically returns `true` for active nodes
