<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getBlockByHash" />


Returns information about a block by hash.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockByHash",
  "params": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    true
  ],
  "id": 1
}
```

### Parameters

1. **blockHash** (`string`, required): Block hash, 32 bytes hex-encoded with `0x` prefix
2. **fullTransactions** (`boolean`, required): If `true`, returns full transaction objects; if `false`, returns only transaction hashes

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "number": "0x1a2b3c",
    "hash": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "parentHash": "0x9646252be9520f6e71339a8df9c55e4d7619deeb018d2a3c2d950ed0f2e5f8e8",
    "transactionsRoot": "0x56e81f171bcc55a6ff8345e692c0f86e5b48e01b996cadc001622fb5e363b421",
    "stateRoot": "0xd5855eb08b3387c0af375e9cdb6acfc05eb8f519e4192fdafd2f764b1b5927a0",
    "miner": "0x0000000000000000000000000000000000000000",
    "difficulty": "0x0",
    "gasLimit": "0x9f759",
    "gasUsed": "0x9f759",
    "timestamp": "0x5ba22882",
    "transactions": [],
    "author": "0x...",
    "boundary": "0x...",
    "seedHash": "0x..."
  }
}
```

### Response Types

- **number** (`string`): Block number, hex-encoded
- **hash** (`string`): Block hash, 32 bytes hex-encoded
- **author** (`string`): Block author (SKALE-specific field)
- **boundary** (`string`): Boundary value (SKALE-specific field)
- **seedHash** (`string`): Seed hash (SKALE-specific field)
- **transactions** (`array`): Array of transaction objects or transaction hashes

## Notes

- Includes SKALE-specific fields: `author`, `boundary`, `seedHash`
- Rotated blocks may not be available on Core/Full Sync nodes
- Available on all node types
- Set `fullTransactions` to `false` for faster responses
