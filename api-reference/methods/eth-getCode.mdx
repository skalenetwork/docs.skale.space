<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getCode" />


Returns code at a given address.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getCode",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "latest"
  ],
  "id": 1
}
```

### Parameters

1. **address** (`string`, required): 20-byte address, hex-encoded with `0x` prefix
2. **block** (`string`, required): Block number or one of `"latest"`, `"earliest"`, `"pending"`

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x608060405234801561001057600080fd5b50..."
}
```

### Response Types

- **result** (`string`): The code at the given address, hex-encoded. Returns `"0x"` if the address is not a contract.

## Notes

- `"latest"` only on Core/Full Sync nodes; Archive nodes support historical blocks
- Supports EIP-1898 block specification
- Returns `"0x"` for EOA (Externally Owned Accounts)
- Available on all node types (with limitations on Core/Full Sync)
