<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_hashrate" />


Returns the number of hashes per second that the node is mining with.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_hashrate",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x0"
}
```

### Response Types

- **result** (`string`): Number of hashes per second as a hexadecimal string

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_hashrate",
    "params": [],
    "id": 1
  }'
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x0"
}
```

</CodeGroup>

## Notes

- Always returns `"0x0"` on SKALE nodes
- Available on all node types (with limitation)
- SKALE uses Proof of Stake consensus, not mining
