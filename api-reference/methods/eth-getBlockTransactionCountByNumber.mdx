<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getBlockTransactionCountByNumber" />


Returns the number of transactions in a block from a block matching the given block number.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockTransactionCountByNumber",
  "params": [
    "0x1a2b3c"
  ],
  "id": 1
}
```

### Parameters

1. **block** (`string`, required): Block number or one of `"latest"`, `"earliest"`, `"pending"`

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5"
}
```

### Response Types

- **result** (`string`): Number of transactions in the block as a hexadecimal string

## Notes

- Old blocks are rotated out on Core/Full Sync nodes (see [Block Rotation](/concepts/block-rotation))
- Available on all node types (with limitations)
- Archive nodes have access to all historical blocks
