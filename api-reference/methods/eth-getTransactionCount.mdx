<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getTransactionCount" />


Returns the number of transactions sent from an address (nonce).

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionCount",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "latest"
  ],
  "id": 1
}
```

### Parameters

1. **address** (`string`, required): 20-byte address, hex-encoded with `0x` prefix
2. **block** (`string`, required): Block number or one of `"latest"`, `"earliest"`, `"pending"`

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5"
}
```

### Response Types

- **result** (`string`): The transaction count (nonce) as a hexadecimal string

```json Response
{
```

</CodeGroup>

## Notes

- `"latest"` only on Core/Full Sync nodes; Archive nodes support historical blocks
- Supports EIP-1898 block specification
- Use this to get the nonce for the next transaction
- Available on all node types (with limitations on Core/Full Sync)
