Creates a new message call transaction or a contract creation.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_sendTransaction",
  "params": [
    {
      "from": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
      "to": "0x3535353535353535353535353535353535353535",
      "value": "0x0",
      "gas": "0x5208",
      "gasPrice": "0x4a817c800",
      "data": "0x"
    }
  ],
  "id": 1
}
```

### Parameters

1. **transaction object** (`object`, required):
   - **from** (`string`, required): Address the transaction is sent from
   - **to** (`string`, optional): Address the transaction is directed to
   - **value** (`string`, optional): Value sent with the transaction in wei
   - **gas** (`string`, optional): Gas limit
   - **gasPrice** (`string`, optional): Gas price in wei
   - **data** (`string`, optional): Hash of the method signature and encoded parameters

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
}
```

### Response Types

- **result** (`string`): The transaction hash, 32 bytes hex-encoded with `0x` prefix

## Notes

- Partial support on Core nodes only
- Personal API is disabled on SKALE nodes
- Use [eth_sendRawTransaction](/api-reference/methods/eth-sendRawTransaction) instead for better compatibility
- Not available on Full Sync or Archive nodes
