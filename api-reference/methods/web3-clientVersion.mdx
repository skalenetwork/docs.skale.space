<Api openapi="api-reference/openapi.json" path="/" method="post" example="web3_clientVersion" />


Returns the current client version.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "web3_clientVersion",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "skaled/v1.x.x"
}
```

### Response Types

- **result** (`string`): The client version string

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{

```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "skaled/v1.x.x"
}
```

</CodeGroup>

## Notes

- Returns the version of the SKALE node software (skaled)
- Available on all node types (Core, Full Sync, Archive)
