<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_blockNumber" />

Returns the number of the most recent block.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1a2b3c"
}
```

### Response Types

- **result** (`string`): The block number as a hexadecimal string (e.g., `"0x1a2b3c"`)

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1a2b3c"
}
```

</CodeGroup>

## Notes

- The result is returned as a hexadecimal string prefixed with `0x`
- To convert to decimal, parse the hex value (e.g., `0x1a2b3c` = 1715004)
- This method is available on all node types (Core, Full Sync, Archive)
