<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_estimateGas" />


Generates and returns an estimate of how much gas is necessary to allow the transaction to complete.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_estimateGas",
  "params": [
    {
      "to": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
      "data": "0x..."
    }
  ],
  "id": 1
}
```

### Parameters

1. **transaction object** (`object`, required):
   - **to** (`string`): 20-byte address, hex-encoded with `0x` prefix
   - **data** (`string`, optional): Hash of the method signature and encoded parameters
   - **from** (`string`, optional): Address the transaction is sent from
   - **gas** (`string`, optional): Gas limit for the call
   - **gasPrice** (`string`, optional): Gas price in wei
   - **value** (`string`, optional): Value sent with the call in wei

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5208"
}
```

### Response Types

- **result** (`string`): The amount of gas used as a hexadecimal string

## Notes

- Does not use binary search (unlike some Ethereum clients)
- Available on all node types
- Use this to estimate gas before sending a transaction
