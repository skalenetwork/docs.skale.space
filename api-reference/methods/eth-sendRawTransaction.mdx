Creates a new message call transaction or a contract creation for signed transactions.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_sendRawTransaction",
  "params": [
    "0xf86c808502540be400825208943535353535353535353535353535353535353535880de0b6b3a76400008025a028ef61340bd939bc2195fe537567866003e1a15d3c71ff63e1590620aa636276a067cbe9d8997f761aecb703304b0200f4d2b6ed7a88599e1b0e3b0e3b0e3b0e3b0e3b"
  ],
  "id": 1
}
```

### Parameters

1. **signedTransactionData** (`string`, required): The signed transaction data as a hex-encoded string

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
}
```

### Response Types

- **result** (`string`): The transaction hash, 32 bytes hex-encoded with `0x` prefix

```json
{
```

## Notes

- The transaction must be signed before sending
- Returns the transaction hash immediately
- Use [eth_getTransactionReceipt](/api-reference/methods/eth-getTransactionReceipt) to check transaction status
- Transaction will be included in the next block (typically within 1 second on SKALE)
- On SKALE, transactions are free (zero gas fees)
