<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_accounts" />


Returns a list of addresses owned by the client.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_accounts",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": []
}
```

### Response Types

- **result** (`array`): Array of account addresses owned by the client

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_accounts",
    "params": [],
    "id": 1
  }'
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": []
}
```

</CodeGroup>

## Notes

- Available on all node types
- Typically returns an empty array for public RPC endpoints
- Only returns accounts if the node has unlocked accounts
