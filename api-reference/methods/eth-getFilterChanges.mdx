Polling method for a filter which returns an array of logs which occurred since last poll.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getFilterChanges",
  "params": [
    "0x1"
  ],
  "id": 1
}
```

### Parameters

1. **filterId** (`string`, required): Filter ID, hex-encoded

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
  ]
}
```

### Response Types

- **result** (`array`): Array of log objects or block/transaction hashes (depending on filter type)

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
```

```json Response
{
```

</CodeGroup>

## Notes

- Available on all node types
- Returns only new items since the last call
- For block filters, returns block hashes
- For transaction filters, returns transaction hashes
- For log filters, returns log objects
