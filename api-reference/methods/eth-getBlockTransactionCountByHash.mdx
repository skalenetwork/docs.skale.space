<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getBlockTransactionCountByHash" />


Returns the number of transactions in a block from a block matching the given block hash.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockTransactionCountByHash",
  "params": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
  ],
  "id": 1
}
```

### Parameters

1. **blockHash** (`string`, required): Block hash, 32 bytes hex-encoded with `0x` prefix

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5"
}
```

### Response Types

- **result** (`string`): Number of transactions in the block as a hexadecimal string

## Notes

- Old blocks are rotated out on Core/Full Sync nodes (see [Block Rotation](/concepts/block-rotation))
- Available on all node types (with limitations)
- Archive nodes have access to all historical blocks
