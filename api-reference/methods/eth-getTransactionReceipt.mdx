<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getTransactionReceipt" />


Returns the receipt of a transaction by transaction hash. The receipt is available only after the transaction has been mined.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionReceipt",
  "params": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
  ],
  "id": 1
}
```

### Parameters

1. **transactionHash** (`string`, required): Transaction hash, 32 bytes hex-encoded with `0x` prefix

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "transactionHash": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "transactionIndex": "0x0",
    "blockHash": "0xfd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2",
    "blockNumber": "0x1a2b3c",
    "from": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "to": "0x3535353535353535353535353535353535353535",
    "gasUsed": "0x5208",
    "cumulativeGasUsed": "0x5208",
    "contractAddress": null,
    "logs": [],
    "logsBloom": "0x0000...",
    "status": "0x1"
  }
}
```

### Response Types

- **transactionHash** (`string`): Hash of the transaction
- **transactionIndex** (`string`): Transaction index position in the block
- **blockHash** (`string`): Hash of the block containing this transaction
- **blockNumber** (`string`): Block number containing this transaction
- **from** (`string`): Address of the sender
- **to** (`string`): Address of the receiver (null for contract creation)
- **gasUsed** (`string`): Gas used by this transaction
- **cumulativeGasUsed** (`string`): Total gas used in the block up to this transaction
- **contractAddress** (`string` | `null`): Contract address if this was a contract creation, otherwise null
- **logs** (`array`): Array of log objects emitted by this transaction
- **logsBloom** (`string`): Bloom filter for logs
- **status** (`string`): `"0x1"` for success, `"0x0"` for failure

## Notes

- Returns `null` if the transaction has not been mined yet
- Use this method to check if a transaction was successful (`status: "0x1"`)
- The receipt is available immediately after the transaction is included in a block
- On SKALE, transactions are typically included within 1 second
