<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getWork" />


Returns the hash of the current block, the seedHash, and the boundary condition to be met.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getWork",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": [
    "0x...",
    "0x...",
    "0x..."
  ]
}
```

### Response Types

- **result** (`array`): Array of three strings:
  1. Current block header pow-hash
  2. Seed hash used for DAG
  3. Boundary condition ("target")

## Notes

- Available on all node types
- Used for Proof of Work (though SKALE uses Proof of Stake)
- Primarily for compatibility with Ethereum tooling
