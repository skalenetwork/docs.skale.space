<Api openapi="api-reference/openapi.json" path="/" method="post" example="eth_getTransactionByHash" />


Returns the information about a transaction requested by transaction hash.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionByHash",
  "params": [
    "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b"
  ],
  "id": 1
}
```

### Parameters

1. **transactionHash** (`string`, required): Transaction hash, 32 bytes hex-encoded with `0x` prefix

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "hash": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "nonce": "0x15",
    "blockHash": "0xfd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2fd43a1c2",
    "blockNumber": "0x1a2b3c",
    "transactionIndex": "0x0",
    "from": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "to": "0x3535353535353535353535353535353535353535",
    "value": "0x0",
    "gasPrice": "0x4a817c800",
    "gas": "0x5208",
    "input": "0x"
  }
}
```

### Response Types

- **hash** (`string`): Transaction hash
- **nonce** (`string`): Transaction nonce
- **blockHash** (`string`): Hash of the block containing this transaction
- **blockNumber** (`string`): Block number containing this transaction
- **from** (`string`): Address of the sender
- **to** (`string`): Address of the receiver (null for contract creation)
- **value** (`string`): Value transferred in wei
- **gasPrice** (`string`): Gas price in wei
- **gas** (`string`): Gas limit
- **input** (`string`): Transaction data

## Notes

- Missing fields: `chainId`, `v`, `r`, `s`, `type` (not included in response)
- Available on all node types
- Returns `null` if transaction not found
