<Api openapi="api-reference/openapi.json" path="/" method="post" example="net_version" />


Returns the current network ID (Chain ID).

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "net_version",
  "params": [],
  "id": 1
}
```

### Parameters

None.

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1"
}
```

### Response Types

- **result** (`string`): Network ID (Chain ID) as a hexadecimal string

## Example

<CodeGroup>

```bash Request
curl -X POST https://mainnet.skalenodes.com/v1/honorable-steel-rasalhague \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "net_version",
    "params": [],
    "id": 1
  }'
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1"
}
```

</CodeGroup>

## Notes

- Returns ChainID from `config.json`
- Available on all node types
- Result is hex-encoded (e.g., `"0x1"` = Chain ID 1)
