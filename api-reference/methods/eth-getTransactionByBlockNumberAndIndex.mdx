Returns information about a transaction by block number and transaction index position.

## Request

```json
{
  "jsonrpc": "2.0",
  "method": "eth_getTransactionByBlockNumberAndIndex",
  "params": [
    "0x1a2b3c",
    "0x0"
  ],
  "id": 1
}
```

### Parameters

1. **block** (`string`, required): Block number or one of `"latest"`, `"earliest"`, `"pending"`
2. **transactionIndex** (`string`, required): Transaction index position in the block, hex-encoded

## Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "hash": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "nonce": "0x15",
    "blockHash": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d28b9e5c6b5b5b5b5b5b5b5b5b",
    "blockNumber": "0x1a2b3c",
    "transactionIndex": "0x0",
    "from": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
    "to": "0x3535353535353535353535353535353535353535",
    "value": "0x0",
    "gasPrice": "0x4a817c800",
    "gas": "0x5208",
    "input": "0x"
  }
}
```

### Response Types

- **hash** (`string`): Transaction hash
- **blockHash** (`string`): Hash of the block containing this transaction
- **blockNumber** (`string`): Block number containing this transaction
- **transactionIndex** (`string`): Transaction index position in the block
- **from** (`string`): Address of the sender
- **to** (`string`): Address of the receiver
- **value** (`string`): Value transferred in wei
- **gasPrice** (`string`): Gas price in wei
- **gas** (`string`): Gas limit
- **input** (`string`): Transaction data

## Notes

- Available on all node types
- Returns `null` if transaction not found
- Transaction index is 0-based
