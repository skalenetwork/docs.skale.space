## Overview

SKALE Network is governed by the SKALE DAO, which allows SKL token holders to participate in network decisions. The SKALE DAO is the decentralized autonomous organization that manages the network.

## Governance Model

- **SKALE DAO** - Decentralized autonomous organization managing the network
- **Proposals** - Community-submitted proposals for network changes
- **Voting** - SKL token holders vote on proposals
- **Execution** - Approved proposals are executed on-chain

## DAO Structure

The SKALE DAO structure includes:

- **DAO Operations** - How the DAO operates and makes decisions
- **DAO Treasury** - Management of network funds
- **DAO Proposals** - Process for submitting and managing proposals

## Participation

To participate in SKALE governance:

- Hold and delegate SKL tokens
- Submit governance proposals
- Vote on active proposals
- Participate in discussions
