### **Programmable Privacy**

SKALE chains have built-in privacy features powered by threshold encryption. Encrypt transaction payloads, build confidential smart contracts, and create tokens with hidden balances — all fully EVM-compatible.

<CardGroup cols={3}>
  <Card title="Encrypted Transactions" icon="lock" href="/concepts/programmable-privacy/encrypted-transactions">
    Hide transaction payloads during mempool and consensus for MEV resistance.
  </Card>

  <Card title="Conditional Transactions" icon="code" href="/concepts/programmable-privacy/conditional-transactions">
    Smart contracts that request decryption on-demand for sealed-bid auctions and voting.
  </Card>

  <Card title="Confidential Tokens" icon="coins" href="/concepts/programmable-privacy/confidential-tokens">
    ERC20 tokens with encrypted balances using dual-layer threshold and ECIES encryption.
  </Card>
</CardGroup>
