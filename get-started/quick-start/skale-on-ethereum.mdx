## Key Features

SKALE on Ethereum testnet offers a variety of features at the chain level such as instant finality, zero gas fees, a native bridge to/from Ethereum, huge block gas limits, and a bespoke high-performance C++ EVM implementation. Additionally, this network boasts ~800 validator nodes run by ~100 SKALE Supernodes powering the multichain network.

<Note>
Interested in privacy features with the power of SKALE? Checkout [SKALE on Base](/get-started/quick-start/skale-on-base) to try out [BITE Protocol](/concepts/bite-protocol/intro-bite-protocol).
</Note>
